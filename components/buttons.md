# Buttons

## Primary
```html preview
<style>
  .button {
    border-radius: 0;
  }
</style>

<button class="button">
  <span>Buy now</span>
</button>
```

## Secondary
```html preview
<style>
  .button {
    border-radius: 0;
  }

  .button-secondary {
    background-color: #000;
    color: #fff;
  }
</style>

<button class="button button-secondary">Add to cart</button>
```

## Full width
```html preview
<style>
  .button {
    border-radius: 0;
  }

  .button-fullWidth {
    display: block;
    width: 100%;
  }
</style>

<button class="button button-fullWidth">Full width</button>
```
